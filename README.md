# Mathematical Foundations for Machine Learning

Exploring statistics and linear algebra concepts through implementation to strengthen mathematical intuition for machine learning.

Machine learning systems rely heavily on mathematical reasoning.  
This repository documents my effort to strengthen the mathematical intuition required to understand data-driven algorithms.

Instead of only studying formulas, I explore how these concepts behave when implemented computationally.

## Areas of Focus

### Statistics
Understanding how data behaves and how uncertainty is measured.

Topics explored include:
- Descriptive statistics
- Probability basics
- Variance and standard deviation
- Correlation and covariance
- Data interpretation through visualization

### Linear Algebra
Exploring the geometric foundations behind machine learning algorithms.

Topics explored include:
- Vectors and vector operations
- Matrix operations
- Dot products
- Linear transformations
- Eigenvalues and eigenvectors (conceptual exploration)

Many ideas explored here are inspired by the *Essence of Linear Algebra* series by 3Blue1Brown combined with practical experimentation in Python.

## Goal

The objective of this repository is to build strong mathematical intuition that will support deeper understanding of machine learning systems and data analysis.
